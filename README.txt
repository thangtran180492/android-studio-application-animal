# android studio
project ứng dụng mobile trên smartphone:
1. mục đích ứng dụng
- ứng dụng cung cấp thông tin và hình ảnh về animal
- thông tin được phân loại của từng nhóm animal

2. chức năng ứng dụng
- ứng dụng có icon tương ứng với mỗi animal
- kích vào mỗi icon sẽ hiển thị thông tin của mỗi animal

- đây là 1 ứng dụng đơn giản trong quá trình học ngôn ngữ lập trình java
